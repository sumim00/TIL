## 리액트 네이티브

### 2. 기본 컴포넌트

| 컴포넌트명 | 설명                                                  |
| ---------- | ----------------------------------------------------- |
| View       | UI 생성에서 가장 기본적인 컴포넌트                    |
| Text       | 텍스트 컴포넌트                                       |
| Image      | 이미지 컴포넌트                                       |
| TextInput  | input 타입을 위한 컴포넌트                            |
| ScrollView | 여러 컴포넌트와 뷰를 호스팅할 수 있는 스크롤 컨테이너 |
| StyleSheet | CSS 스타일 설정을 위한 컴포넌트                       |
