#### 특징

- 정적 언어
  - 코드 수준에서 미리 타입을 체크하기 때문에 빠르게 오류를 발견할 수 있다.
  - JavaScript = 동적 타이핑 (변수나 반환값의 타입을 사전에 지정하지 않음)
- Compiled Language
  - 컴파일러를 거치는 언어이기 때문에 컴파일 단계에서 타임 오류를 잡을 수 있다.
  - JavaScript = Interpreted Language (컴파일러를 거치지 않고 바로 실행되는 언어)
